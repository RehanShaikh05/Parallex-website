# Parallax Adventure Website

This is a simple Parallax scrolling website built using HTML and CSS.  
The project demonstrates how to create depth and scrolling effects using CSS transforms and background images.

## Features
- Parallax scrolling effect
- Adventure themed sections
- Smooth animations

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
